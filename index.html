📊 EXECUTIVE SUMMARY
Updated June 2026

SummerStreak is a gamified 12-week summer reading app for elementary-school kids, currently in a small-family beta. Built in a long weekend by a non-engineer parent using AI-assisted "vibe coding." Live at beatthesummerslide.com. Target outcome: measurable improvement in fall 2026 MAP RIT scores vs. spring baselines.

💡 Troubleshooting tips

Open the dashboard in your browser → press F12 (or Cmd+Opt+I on Mac) → Application tab → Local Storage → declan-dashboard-v3. That's the full state. Paste it back at you (or me) to diagnose anything.
If a user is stuck, ask them: "What's the URL in your address bar?" The bit after # tells you which tab. Add ?parent=1 to auto-expand the parent section.
